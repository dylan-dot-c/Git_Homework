What is HTML? | Main

HTML is a simple markup language that is used to build the structure of a webpage. HTML means Hyper Text Markup Language. This is so because it is basically a markup language that is used to structure information on a webpage.

What is CSS? | Feature

CSS also known as Cascading Style Sheets is a language that is used to provide your own custom stylying on your webpage.

Css can be used to select and individual or a group of components and apply specific rules defined by the programmer. Css attributes are in the form of attribute: value;
